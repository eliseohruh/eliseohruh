# Elise Ohruh - Sophrologue

Site web professionnel pour Elise Ohruh, sophrologue à Surbourg.

## 📋 Fonctionnalités

- Page d'accueil avec présentation
- Section "À propos"
- Accompagnements proposés
- Tarifs
- Contact
- Certification

## 🚀 Installation

1. Clonez le repository
2. Installez les dépendances :
```bash
npm install
```

3. Lancez le serveur de développement :
```bash
npm run dev
```

4. Ouvrez [http://localhost:3000](http://localhost:3000) dans votre navigateur

## 📝 Images requises

Placez vos images dans le dossier `public/` :
- `Logo_Elise_FINAL.png`
- `Fleur.png`
- `diplome.png`

## 🌐 Déploiement sur Vercel

1. Connectez votre repository GitHub à Vercel
2. Vercel détectera automatiquement que c'est un projet Next.js
3. Cliquez sur "Deploy"
